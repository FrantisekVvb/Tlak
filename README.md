# Tlak

Interaktivní simulace tlaku.

**Online verze:** https://frantisekvvb.github.io/Tlak/

## Spuštění lokálně

Potřebuješ [Node.js](https://nodejs.org/) 18 nebo novější.

```bash
npm start
```

Aplikace poběží na adrese **http://localhost:3475**.

Alternativa bez Node.js:

```bash
python3 -m http.server 3475
```
